# Analysis Methodology

## Approach

1. **Portfolio Health Assessment**
   - Portfolio at Risk (PAR) calculations
   - Collection rate analysis
   - Loan status distribution

2. **Risk Segmentation**
   - Geographic (7 regions)
   - Borrower behavior (loan cycles 1-12)
   - Business type (5 sectors)
   - Vintage (monthly cohorts)

3. **Statistical Validation**
   - Sample sizes verified
   - 95% confidence intervals
   - Effect size measurements

## Key Assumptions

- Recovery rate: 18% on defaulted loans
- Officer capacity: 100-150 loans optimal
- Portfolio growth: 15% annually in high-performing regions
